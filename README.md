# House Price Predictor ML

A machine learning project that predicts house prices using Python.

## Features

- Loads housing data from a CSV file
- Trains a machine learning model
- Predicts house prices
- Evaluates model performance

## Technologies Used

- Python
- Pandas
- Scikit-learn

## Project Structure

house-price-predictor-ml

├── main.py  
├── house_data.csv  
├── requirements.txt  
└── README.md  

## How to Run

1. Install dependencies:

pip install -r requirements.txt

2. Run the program:

python main.py

## Machine Learning Model

This project uses Linear Regression to learn the relationship between:

- House area
- Number of rooms
- House age

and predict:

- House price
